Tela de Cadastro

Uma tela de cadastro, onde você pode se registrar como doador ou receptor de sangue, informando seus dados pessoais, tipo sanguíneo, histórico médico, etc.
Para essa tela, eu sugiro usar um fundo branco para dar uma sensação de limpeza e clareza.
No topo da tela, eu sugiro repetir o mesmo logo e o mesmo menu da tela anterior, para manter a consistência visual do site.
No centro da tela, eu sugiro colocar um formulário dividido em duas partes: uma para os dados pessoais e outra para os dados médicos.
Na parte dos dados pessoais, eu sugiro pedir o nome completo, o e-mail, a senha, o telefone e o endereço do usuário, usando campos de texto ou caixas de seleção.
Na parte dos dados médicos, eu sugiro pedir o tipo sanguíneo, o peso, a altura, as alergias, as doenças crônicas e as medicações do usuário, usando campos de texto ou caixas de seleção.
Eu sugiro também incluir uma caixa de seleção que permite que o usuário escolha se quer se cadastrar como doador ou receptor de sangue.
Eu sugiro também incluir uma caixa de seleção que permite que o usuário aceite os termos e condições do site.
Abaixo do formulário, eu sugiro colocar um botão vermelho que permite que o usuário confirme o seu cadastro.
